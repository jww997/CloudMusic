### 目录结构

```
mc
├─ README.md
├─ package.json
├─ index.html
├─ public          # 静态资源
├─ src             # 源码
│  ├─ components   # 公共组件
│  ├─ config       # 配置
│  ├─ constant     # 常量
│  ├─ layout       # 布局
│  ├─ locale       # 国际化
│  ├─ plugins      # 插件
│  ├─ router       # 路由
│  ├─ store        # 状态管理
│  ├─ styles       # 全局样式
│  ├─ types        # 类型声明
│  ├─ utils        # 工具库
│  └─ views        # 页面模板
├─ tsconfig.json   # 配置编译
└─ vue.config.js   # 配置脚手架
```
